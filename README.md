# Movie Explorer

🎬 Una app de películas tipo Stremio usando React y Vite.  
- Buscar películas y ver resultados en tiempo real (debounce).  
- Categorías: Trending, Top Rated, Now Playing.  
- Modal con trailer de YouTube y descripción.  
- Favoritos guardados en localStorage.  

🔗 [Ver online](https://movie-explorer.vercel.app)
